住宅 IP 出站

OpenAI / Google / Claude 流量走住宅 IP，其余流量直出 VPS。
支持格式: socks5://user:pass@host:port 或 host:port:user:pass
分流域名设置
每行一个根域名（如 openai.com），自动匹配该域名及所有子域名

伪装网站设置

此设置将同时应用于 Hysteria2 和 VLESS-Reality 协议，作为流量伪装目标。

全局带宽限制

设置 Hysteria2 服务的全局带宽限制，应用于所有客户端。每个客户端独立享有此带宽配额。
说明：
• 单位换算：100 Mbps = 12.5 MB/s
• 设置为 0 表示无限制
• 每个客户端独立享有此配额，不是共享总带宽

端口跳跃设置

端口跳跃可以有效对抗 QoS 限速，通过在多个端口间跳跃来规避运营商的流量限制。
说明：
• 推荐端口范围：20000-30000
• 需确保防火墙已开放对应 UDP 端口
• 修改后订阅链接会自动包含 mport 参数
